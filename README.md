# movie-app

## Step to run it
- Create a .env file on the root of the project
- Add this to it
```
API_URL=https://api.themoviedb.org/3
API_KEY=cfa9e531766dbc62c39e767373bd05c7
```
- `yarn install`
- `npx pod-install ios`

- run ios: `yarn ios`
- run android: `yarn android`
